# LinkedIn Post Scraper (Chrome Extension)

> [!WARNING]
> **ARCHIVED AND UNMAINTAINED**
> This code is no longer maintained and remains public purely as a historical portfolio archive.
>
> *Note: Due to frequent DOM changes and anti-scraping measures implemented by LinkedIn, this extension likely no longer functions as intended.*

A Google Chrome extension built in JavaScript (using Manifest V3) designed to scrape LinkedIn posts directly from the browser and output the extracted data into a clean JSON format.
